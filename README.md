## Projetos em destaque

### Sistema de Atendimentos

Sistema web para gerenciamento de filas, senhas, guichês e atendimentos presenciais. Inclui emissão pública de senhas, painel de chamadas, dashboard administrativo, regras de prioridade e fluxo completo de atendimento.

**Stack:** Laravel, Vue.js, Inertia.js, Tailwind CSS, MySQL, Pest, PHPStan e GitHub Actions.

### Sistema de Simulados

Aplicação web para criação, publicação e resposta de simulados online. Possui área administrativa, controle de papéis e permissões, tentativas por usuário, cálculo de nota, relatórios e regras de negócio para validação de perguntas e respostas.

**Stack:** Laravel, Vue.js, Inertia.js, Tailwind CSS, MySQL, Spatie Permission e Laravel Fortify.

### Sistema RSC-TAES

Sistema para solicitação e acompanhamento de processos de Reconhecimento de Saberes e Competências para servidores técnico-administrativos.

**Stack:** Laravel, Vue.js, Inertia.js, Tailwind CSS e MySQL.
